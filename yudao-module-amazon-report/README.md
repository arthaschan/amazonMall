# yudao-module-amazon-report

## Amazon Reports & AI Assistant (M8)

Sales dashboards, profit/loss reports, advertising analytics, keyword ranking reports, business insights, and AI-powered seller assistant.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
