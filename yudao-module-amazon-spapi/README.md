# yudao-module-amazon-spapi

Amazon Selling Partner API client library.

## Sub-modules

| Module | Description |
|--------|-------------|
| `spapi-client` | Generated client stubs for all SP-API endpoints (Orders, Catalog, Listings, Reports, Finances) |
| `spapi-auth` | LWA OAuth 2.0 authentication: token refresh, credential management, IAM/STS |
| `spapi-ads` | Amazon Advertising API client: Sponsored Products/Brands/Display, DSP |
