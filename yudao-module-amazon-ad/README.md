# yudao-module-amazon-ad

## Amazon Advertising Management (M6)

Sponsored Products/Brands/Display campaign management, bid optimization, budget management, keyword/targeting management, ACoS tracking, and report generation.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
