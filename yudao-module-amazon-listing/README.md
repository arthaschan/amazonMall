# yudao-module-amazon-listing

## Amazon Listing Management (M3)

Listing creation and optimization, title/bullet/description/backend-keywords editing, A+ content management, variation management, and bulk upload.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
