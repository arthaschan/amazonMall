# yudao-module-amazon-research

## Amazon Product Research & Selection (M2)

Product opportunity scouting, niche analysis, BSR tracking, competitor benchmarking, keyword research, demand estimation, and profitability analysis.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
