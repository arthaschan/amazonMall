# yudao-module-amazon-order

## Amazon Order & Logistics (M4)

Order sync and tracking, FBA shipment management, shipping label generation, return/refund handling, and multi-marketplace order aggregation.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
