# yudao-module-amazon-shop

## Amazon Store/Account Management (M1)

Seller account binding, marketplace configuration, store profile management, credential storage, and multi-marketplace account switching.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
