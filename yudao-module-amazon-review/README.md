# yudao-module-amazon-review

## Amazon Review & Customer Service (M7)

Review monitoring and alerts, review analytics, buyer-seller messaging, feedback management, A-to-Z claim handling, and customer service automation.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
