# yudao-module-amazon-ai

AI-powered features for Amazon seller operations.

## Sub-modules

| Module | Description |
|--------|-------------|
| `amazon-ai-core` | Prompt templates, model router, token usage tracking, embedding management |
| `amazon-ai-listing` | Listing generation/diagnosis AI: title, bullets, description, A+ content |
| `amazon-ai-review` | Review analysis AI: sentiment, topics, pain points, VOC intelligence |
| `amazon-ai-research` | Product selection analysis AI: market opportunity, competition, profitability |
| `amazon-ai-ad` | Ad optimization AI: bid suggestions, budget allocation, ACoS optimization |
| `amazon-ai-inventory` | Inventory forecast AI: demand prediction, reorder points, seasonal trends |
| `amazon-ai-agent` | Chat assistant Agent with Function Calling: tool orchestration, multi-step tasks |
