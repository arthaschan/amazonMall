# yudao-module-amazon-inventory

## Amazon Inventory Management (M5)

FBA inventory monitoring, stock level tracking, reorder alerts, IPI score optimization, stranded inventory recovery, and multi-warehouse coordination.

## Package Structure

- `controller/admin` - REST API controllers
- `service` - Business logic
- `dal/dataobject` - Data objects (DOs)
- `dal/mysql` - MyBatis-Plus mapper interfaces
- `framework/web/config` - Swagger and web configuration
